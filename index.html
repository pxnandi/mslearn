HTML

Header

Для подключения стилей к странице существует тег <link>. Для этого у него есть атрибут href в котором задаётся адрес стилевого файла, а значение stylesheet атрибута rel говорит браузеру, что мы подключаем именно стили, а не что-то другое. 

            
  <head>
    <link href="адрес_файла_стилей.css" rel="stylesheet">
  </head>

Ещё один элемент, который располагается в <head> — это тег <title>. В нём задаётся заголовок страницы, который отображается во вкладках браузера. 

            
  <head>
    <title>Веб конспект</title>
  </head>

Meta

Ещё один важный тег, располагающийся внутри <head> это тег <meta>. С помощью <meta> можно сообщать браузеру, поисковому роботу или другому устройству различную служебную информацию (или метаинформацию) о вашем сайте: кодировку текста, описание контента и так далее.

Кодировка текста HTML-страницы указывается с помощью атрибута charset:

<meta charset="название кодировки">

Самая распространённая современная кодировка — utf-8.

Перечень ключевых слов задаётся тегом <meta>, у которого атрибут name имеет значение keywords. Ключевые слова (самые важные слова из содержания страницы) перечисляются в атрибуте content через запятую:

<meta name="keywords" content="важные, ключевые, слова">

Краткое описание (или аннотация) страницы задаётся похожим образом, только значение атрибута name меняется на description:

<meta name="description" content="краткое описание">

Теги разметки документа

<main> используется для вывода основного содержания страницы.

<header> содержит вводную информацию страницы (шапка страницы).

<footer> содержит заключительную информацию страницы (подвал страницы).

<section> указывает на крупный смысловой или логический раздел.

<article> обозначает цельный, законченный и самостоятельный фрагмент информации.

<nav> (navigation) отвечает за навигацию, как правило включает в себя ссылки на другие страницы или навигацию по текущей.

<aside> содержит дополнительную, никак не связанную с основным содержанием информацию (сайдбар/боковая панель).

Теги от <h1>; до <h6> используются для создания заголовков, h1 - самый старший заголовок, h6 - самый младший подзаголовок.

<p> (paragraph) создаёт параграф, более мелкую структуру текста.


Списки

<ul> (unsorted list) используется для создания неупорядоченного списка.

<li> (list item) является элементом списка.

<ol> (ordered list) используется для создания упорядоченного списка, может иметь несколько атрибутов: start, reversed и type.

  • start меняет стартовое число нумерации пунктов.
  • reversed меняет направление нумерации на противоположное.
  • type может задавать различные типы маркеров: строчные и заглавные латинские буквы или римские цифры.

<dl> (description list) создаёт список описаний, который задаётся с помощью тегов: <dl>, <dt> и <dd>.
<dt> (description term) обозначает термин.
<dd> (description definition) обозначает описание или определение.


Преформатированный текст и код

<pre> (preformatted text) сохраняет все пробелы внутри тега.

<code> используется для обозначения фрагментов кода, можно вкладывать внутрь <pre>.


Цитаты

<q> (quote) редназначен для выделения цитат внутри предложения.

В теге <cite> [источник цитат] можно указывать помимо адреса источника цитаты ещё и название произведения, откуда цитируется текст, а также имя автора или организации, чей текст цитируется.

<blockquote> предназначен для выделения длинных цитат, которые могут состоять из нескольких абзацев.


Разметка фрагментов текста

Символы-мнемоники — это особые строки. Например, знак меньше на страницу можно вставить мнемоникой &lt; (less than), а знак больше мнемоникой &gt; (greater than).

<br> (line break) создаёт перенос строки.

<sup> (superscript) отображает текст в виде верхнего индекса.
<sub> (subscript) отображает текст в виде нижнего индекса.

<time> используется для показания времени, используется с атрибутом datetime, внутри которого пишется время для машины в виде 2012-01-24T00:00.

<em> (emphasis) выделяет текст курсивом, используется при определении текста, на который сделан особый акцент, меняющий смысл предложения, например ирония.
<i> (italic) выделяет текст курсивом, используется для обозначения текста, который отличается от окружающего текста, но не является более важным, например для терминов или названий.

<strong> выделяет текст жирным, указывает на важность отмеченного текста, может использоваться для выделения предупреждений или части документа, которую пользователь должен увидеть раньше остального.
<b> (bold) выделяет текст жирным, предназначен для выделения текста с целью привлечения к нему внимания, но без придания ему особой важности.

<del> (delete) текст перечёркивается, выделяет текст, который был удалён в новой версии документа.
<ins> (insert) текст подчёркивается, выделяет текст, который был добавлен в новой версии документа.

<div> используется для группировки структурных элементов или в качестве вспомогательных контейнеров для создания нужной раскладки.
<span> используется для группировки текстовых элементов, выделения отдельных слов или фраз внутри абзацев, пунктов списка и так далее.


Ссылки

<a> (anchor) создаёт ссылку на странице, ссылка указывается в атрибуте href (hyper reference).

Для скаивания файла существует атрибут download.

При скачивании или загрузке файлов со сторонних сайтов для безопасности можно использовать атрибут rel="noopener". Этот атрибут позволяет игнорировать скрипты сторонней страницы, которые могут влиять на загрузку файла.

Тег <a> можно использовать вообще без атрибута href, такой тег обозначает ссылку-заглушку.

При помощи атрибута title можно добавить подсказку, которая будет появляться когда когда курсор задержится над ссылкой.

Ссылка-якорь — это обычная ссылка, в адресе которой используется символ #, после которого следует id элемента. Id создаётся с помощью атрибута id у того тега, к которому надо перейти при щелчке по ссылке.


Изображения

<img> (image) добавляет изображение на страницу, путь к изображению указывается в атрибуте src (source).
Атрибуты width и height используются для управления шириной и высотой изображени.я
Атрибут alt позволяет добавить альтернативный текст, который будет показываться в случае, если картинка не отображается.

Формат SVG переводится как масштабируемая векторная графика. Качество таких изображений не меняется при изменении размеров, да и вес у них небольшой. Отлично подходит для малоцветных схем, логотипов и иконок.
Формат JPEG подходит для фотографий, рисунков с большим количеством разноцветных деталей, изображений с плавным переходом яркости и контраста. При сжатии изображения ухудшается его качество.
Формат PNG позволяет сохранять изображения, в которых требуется особенная чёткость. Подходит для изображений с прозрачностью и полупрозрачностью.
Формат GIF используется для простейших анимаций.

<figure> подходит для любого иллюстративного или демонстрационного материала, которым можно дополнить содержание документа: схемы, графики, примеры кода, таблицы.
<figcaption> добавляет пояснительный комментарий для <figure>, размещается первым или последним элементом внутри <figure>.

CSS

Теория CSS

CSS-правило состоит из селектора и перечня свойств и их значений:


  селектор {
    свойство: значение;
    свойство: значение;
  }

Если у CSS-правил отличаются только селекторы, а свойства и значения одинаковые, то их можно сгруппировать через запятую.

Можно комбинировать любые типы селекторов через пробел. Такие селекторы называются вложенными или контекстными и читаются справа налево:


  nav a {…}
  .menu ul {…}
  .post .title {…}

Наследование в CSS — это механизм, с помощью которого значения свойств элемента-родителя передаются его элементам-потомкам. Стили, присвоенные одному элементу, наследуются всеми потомками (вложенными элементами), но только в том случае, если они где-то явно не переопределены.

В CSS есть обычные свойства, управляющие одним параметром отображения, и есть составные свойства, управляющие одновременно несколькими параметрами. Например, свойство font. Оно задаёт сразу шесть параметров: размер и название шрифта, высоту строки и некоторые другие.

font: 16px/26px "Arial", sans-serif;

Если значение обычного свойства не было задано в составном, то браузер при отображении страницы использует исходное значение этого свойства.

Внешние стили подключаются из файла и при помощи <link>.
Встраивание стилей в тег <style>, обычно размещют внутри <head>, такой способ обычно используется для оптимизации страницы, ведь в таком случае браузер не будет отправлять дополнительных запросов на сервер.
Встраивание стилей в атрибут style, свойства и значения прописанные таким образом применяются точечно к одному элементу.


Оформление текста

font-size — размер шрифта.

Величина 1em — такой же размер шрифта, что и у родителя. Соответственно, если мы хотим, чтобы шрифт дочернего элемента был всегда в 2 раза больше родительского, то зададим значение 2em.

line-height — высота строки или межстрочный интервал, по умолчанию стоит значение normal, которое равняется 100%-120% от размера шрифта.
Также можно задать множитель: line-height: 150%; — высота строки будет равна 150% от размера шрифта.

font-family — семейство шрифта, в значении указывается список названий шрифтов, перечисленных через запятую. В начале списка располагают самый желаемый шрифт, а в самом конце списка — общий тип шрифта. Браузер проходит по списку слева направо и использует первый найденный в системе или на сайте шрифт.

  • serif — шрифт с засечками
  • sans-serif — шрифт без засечек
  • monospace — моноширинный шрифт
  • cursive — шрифт с неформальным начертанием, например, имитация рукописного текста или леттеринга
  • fantasy — декоративный шрифт, например, всемирно известный Comic Sans

font-weight — насыщенность шрифта, основные значения это normal и bold, но может также принимать значение одного из девяти числовых вариантов:

  • 100: Thin
  • 200: Extra Light (Ultra Light)
  • 300: Light
  • 400: Normal
  • 500: Medium
  • 600: Semi Bold (Demi Bold)
  • 700: Bold
  • 800: Extra Bold (Ultra Bold)
  • 900: Black (Heavy)

text-align — горизонтальное выравнивание текста (и некоторых других элементов), принимает следующие значения:

  • left — выравнивание по левому краю блока, значение по умолчанию
  • right — по правому краю блока
  • center — по центру блока
  • justify — по ширине блока

vertical-align — вертикальное выравнивание текста (и некоторых других элементов), принимает следующие значения:

  • top — выравнивание по верхнему краю строки
  • middle — по середине строки
  • bottom — по нижнему краю строки
  • baseline — по базовой линии строки, значение по умолчанию

color — цвет текста.
background-color — цвет заднего фона.
background-image — картинка заднего фона, в значении в кавычках записывается путь к картинке - url("").
Цвет можно записать при помощи текста: black, white, red; при помощи rgb: rgb(155, 255, 0); при помощи шестнадцатиричного формата: #e2e4ff.
Помимо шести знаков, цветовой код может содержать три, в этом случае второй символ в цветовых составляющих дублируется первым.
rgb имеет расширенную версию с поддержкой непрозрачности rgba, непрозрачность записывается числом от 0 до 1.

padding — отступы внутри блока.
margin — отступы снаружи блока.

white-space — управление пробелами, может принимать следующие значения:

  • nowrap — схлопывает лишние пробелы и отображает весь текст одной строкой без переносов
  • pre — сохраняет пробелы и переносы как в исходном коде аналогично тегу <pre>
  • pre-wrap — работает как значение pre, но добавляет автоматические переносы, если текст не помещается в контейнер
  • normal — режим по умолчанию: лишние пробелы и переносы строк схлопываются, текст переносится, пробелы в конце строк удаляются
  • break-spaces — поведение идентично pre-wrap, но при схлопывании пробелов остаются нетронутыми зарезервированные пробелы, такие как пробелы в конце строки
  • pre-line — как normal, лишние пробелы схлопываются, но строки переносятся на символе переноса строки, на <br> и по мере заполнения строки

text-decoration — дополнительное оформление текста, можно указать несколько значений, указав их через пробел, значения:

  • underline — подчёркивание
  • line-through — зачёркивание
  • overline — надчёркивание
  • none — убирает вышеперечисленные эффекты

font-style — задаёт начертание текста, значения:

  • normal — обычное начертание
  • italic — курсивное начертание
  • oblique — наклонное начертание

Курсив от наклона отличается тем, что сли задано значение italic, браузер будет пытаться найти в заданном шрифте отдельное курсивное начертание символов, а если oblique, то браузер сымитирует курсив, наклонив текст.

text-transform — регистр символов, значения:

  • lowercase — все строчные
  • uppercase — все заглавные
  • capitalize — каждое слово начинается с большой буквы
  • none — отменяет изменение регистра

JavaScript

Основы

console.log('команда'); — вывод чего-либо в консоль.

элемент.querySelector('имя_селектора'); — поиск селектора в документе.

элемент.classList.remove('имя_класса'); — удаление класса.

элемент.classList.add('имя_класса'); — добавление класса.

let имя_переменной = значение_переменной; — объявление переменной.
В JS имя переменной не должно начинаться с цифры, из спецсимволов разрешены только _ и $, также имя переменной чувствительно к регистру.

элемент.onclick = function() { #код }; — при нажатии на элемент выполнить #код.

элемент.classList.toggle('класс');
Если класс у элемента есть, метод classList.toggle ведёт себя как classList.remove и класс у элемента убирает. А если указанного класса у элемента нет, то classList.toggle, как и classList.add, добавляет элементу этот класс.

textContent — параметр, обозначающий текст элемента.

элемент.value — значение элемента.

onsubmit — свойство (обработчик событий), обрабатывающее отправку формы.

Конкатенация в JS записывается с помощью знака +.

Условия и создание элементов

variable += 2 -- увеличивает переменную на 2

variable -= 2 -- уменьшает переменную на 2

variable++ -- увеличивает переменную на 1

variable-- -- уменьшает переменную на 1

элемент.classList.contains('класс'); -- преверяет есть ли у элемента класс

Условная конструкция if else

  if (условие) {
    // Инструкции, которые выполнятся, если условие истинно
  } else {
    // Инструкции, которые выполнятся, если условие ложно

элемент-родитель.append(добавляемый-элемент); -- добавляет указанный в скобках элемент в конец элемента-родителя

document.createElement('имя тега'); -- создаёт элемент на странице

value -- указывает на значение элемента
